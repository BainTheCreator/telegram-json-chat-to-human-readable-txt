# Парсер из .json файла отката чата в человекочитаемый txt файл

Конвертирует экспорт чата (JSON) в удобный для чтения текстовый формат.

## Инструкция по применению

1. **Подготовка:** Создайте папку `result` в той же папке, где лежит исполняемый файл (`.exe` или бинарник).
2. **Исходные данные:** Положите файл экспорта чата с именем `result.json` в папку `data` (путь относительно программы: `data/result.json`).
3. **Запуск:** Запустите программу.
4. **Результат:** Готовый человекочитаемый текст появится в папке `result` в файле `result.txt`.
