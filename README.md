# Чат-ридер

Конвертирует экспорт чата (JSON) в человекочитаемый текстовый файл.

## Что нужно для работы

Рядом с программой (.exe или бинарником для Linux/macOS) должна быть папка **`data`** с файлами экспорта чата в формате **`.json`**.

Запустите программу — она покажет список JSON-файлов (если их несколько), попросит ввести имя папки и имя результирующего .txt файла. Результат появится в папке `result`.
